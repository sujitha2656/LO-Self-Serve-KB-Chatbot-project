# Configuring Automatic Email Forwarding Rules

## Problem
Users want to automatically copy/route emails from specific clients to other team members or internal mailboxes.

## Resolution
1. Open Microsoft Outlook for desktop or go to `https://outlook.office.com`.
2. Click **Settings** (gear icon) > **Mail** > **Rules**.
3. Click **Add new rule** and name it.
4. Define the condition:
   - Example: *If the message sender is `client@partner.com`*
5. Define the action:
   - Select *Forward to* or *Redirect to*.
   - Enter the target colleague's corporate email.
6. Check the box "Stop processing more rules" if this is a priority filter.
7. Click **Save**.
