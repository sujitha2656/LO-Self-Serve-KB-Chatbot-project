# Installing Approved Software via Company Portal

## Problem
Employees need standard business tools (e.g. Slack, Zoom, Notepad++) but lack local administrator rights to install setup files.

## Resolution
1. Press the **Windows Key** on your keyboard, type **Company Portal**, and press Enter.
2. If using macOS, open **Self Service** from the Applications folder.
3. Browse the home catalog of approved software packages.
4. Click on the desired software (e.g., **Slack Desktop**).
5. Click **Install**.
6. The installer runs silently in the background with administrative privileges. Once the status changes to "Installed", you can launch the app from your start menu.
